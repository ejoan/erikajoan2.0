Rubicon Enterprise Bidder: Interface Enhancements
The Rubicon Project Enterprise Bidder (RPEB) is a self-serve bidding platform that was built by developers, but without guidance from a designer. This resulted in a dated looking product, with no branding that was hard to use. Although it hasn't gone through a complete transition, there are some small improvements that have created a much more enjoyable experience.
1. Navigation
Adding brand colours and a context appropriate logo was the first of many changes to create big visual impact on the Enterprise Bidder.

  • Before
2. Buttons
The button updates included creating copy consistency, removal of icons, and a style change to bring them up to date.
  • Outcome
  • Before
3. Reports
Similar to the older buttons, the report titles included icons that were not necessary. Hamburger icons used to sit to the left of the report title for no apparent reason. The report title now has breathing room, statuses are colour-coded for easy reading, column titles are lower case, and the highlighted row colour is easier on the eyes.
